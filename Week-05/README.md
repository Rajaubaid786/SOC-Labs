## 📖 Overview

During Week 5 of the Cyberster Blue Team Internship, I simulated and investigated a Zeus malware infection by creating custom detection rules, identifying Indicators of Compromise (IOCs) and Indicators of Attack (IOAs), mapping attacker behavior to the MITRE ATT&CK framework, and developing an Incident Response Plan following the NIST SP 800-61 lifecycle.

---

## 🎯 Objectives

- Simulate a Zeus malware infection
- Create custom detection rules in Wazuh and Suricata
- Identify IOCs and IOAs
- Perform threat hunting activities
- Map attacker techniques using MITRE ATT&CK
- Develop an Incident Response Plan

## 📖 Overview

During Week 3 of the Cyberster Blue Team Internship, I expanded the SOC lab by integrating network security monitoring using Suricata IDS and Linux firewall management. The primary objective was to detect network-based threats, analyze alerts, and correlate network events with host-based telemetry from Wazuh.

---

## 🎯 Objectives

- Deploy and configure Suricata IDS
- Configure firewall rules using iptables
- Monitor network traffic
- Correlate Suricata alerts with Wazuh events
- Understand host-based vs network-based monitoring
